# Dailsap

A web3-based front-end application developed on the Solana network

## Run Locally

Clone the project

```bash
  git clone https://github.com/tolgaand/dailsap
```

Go to the project directory

```bash
  cd dailsap
```

Install dependencies

```bash
  npm install
  //or
  yarn install
```

Rename `.env.local-example` to `.env.local`

Start the project

```bash
  npm run dev
  //or
  yarn dev
```
